# 🤖 Werjo Bot - Discord Love Bot

بوت ديسكورد مصمم لنشر المحبة والإيجابية في السيرفر من خلال الرسائل الصباحية والمسائية ورسائل التشجيع.

## ✨ المميزات

- 🌅 **رسائل صباحية تلقائية** - كل يوم في الساعة 8:00 صباحاً
- 🌙 **رسائل مسائية تلقائية** - كل يوم في الساعة 8:00 مساءً  
- 💕 **رسائل محبة عشوائية** - كل ساعتين تقريباً
- 🎉 **ترحيب بالأعضاء الجدد** - رسالة ترحيب جميلة لكل عضو جديد
- 🎨 **تصميم أنيق** - استخدام Embeds ملونة وجذابة
- 📱 **أوامر تفاعلية** - أوامر يمكن للأعضاء استخدامها

## 🛠️ التثبيت والإعداد

### 1. متطلبات النظام
```bash
Python 3.8+
pip (مدير حزم Python)
```

### 2. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 3. إعداد البوت في Discord

1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. أنشئ تطبيق جديد (New Application)
3. اذهب إلى قسم "Bot" وأنشئ بوت جديد
4. انسخ التوكن (Token)
5. في قسم "OAuth2 > URL Generator":
   - اختر "bot" في Scopes
   - اختر الصلاحيات التالية في Bot Permissions:
     - Send Messages
     - Use Slash Commands
     - Embed Links
     - Read Message History
     - Use External Emojis

### 4. إعداد ملف البيئة

قم بتعديل ملف `.env`:
```env
DISCORD_TOKEN=your_bot_token_here
GENERAL_CHANNEL_ID=your_channel_id_here
TIMEZONE=Africa/Cairo
```

**للحصول على Channel ID:**
1. فعّل Developer Mode في Discord (User Settings > Advanced > Developer Mode)
2. انقر بالزر الأيمن على القناة واختر "Copy ID"

### 5. تشغيل البوت
```bash
python bot.py
```

## 📋 الأوامر المتاحة

| الأمر | الوصف | مثال |
|-------|--------|-------|
| `!صباح` | رسالة صباحية | `!صباح` |
| `!مساء` | رسالة مسائية | `!مساء` |
| `!حب` | رسالة محبة | `!حب` |
| `!تشجيع` | رسالة تشجيع | `!تشجيع` |
| `!مساعدة` | قائمة الأوامر | `!مساعدة` |
| `!احصائيات` | إحصائيات السيرفر (للمشرفين) | `!احصائيات` |

## ⚙️ التخصيص

### تغيير أوقات الرسائل
في ملف `config.py`:
```python
MORNING_TIME = "08:00"  # الساعة 8 صباحاً
EVENING_TIME = "20:00"  # الساعة 8 مساءً
```

### إضافة رسائل جديدة
في ملف `messages.py`، يمكنك إضافة رسائل جديدة إلى القوائم:
- `MORNING_MESSAGES` - رسائل الصباح
- `EVENING_MESSAGES` - رسائل المساء  
- `LOVE_MESSAGES` - رسائل المحبة
- `ENCOURAGEMENT_MESSAGES` - رسائل التشجيع

### تغيير الألوان
في ملف `config.py`:
```python
COLORS = {
    'morning': 0xFFD700,  # ذهبي للصباح
    'evening': 0x9932CC,  # بنفسجي للمساء
    'love': 0xFF69B4,     # وردي للمحبة
    'success': 0x00FF00,  # أخضر للنجاح
    'info': 0x00BFFF      # أزرق للمعلومات
}
```

## 🔧 استكشاف الأخطاء

### البوت لا يتصل
- تأكد من صحة التوكن في ملف `.env`
- تأكد من أن البوت مدعو للسيرفر بالصلاحيات المطلوبة

### الرسائل لا تُرسل تلقائياً
- تأكد من صحة `GENERAL_CHANNEL_ID` في ملف `.env`
- تأكد من أن البوت لديه صلاحية الكتابة في القناة

### خطأ في المنطقة الزمنية
- تأكد من صحة `TIMEZONE` في ملف `.env`
- قائمة المناطق الزمنية المدعومة: [pytz timezones](https://gist.github.com/heyalexej/8bf688fd67d7199be4a1682b3eec7568)
