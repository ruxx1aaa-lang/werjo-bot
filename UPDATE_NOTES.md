# 🆕 تحديث Werjo Bot - إدارة القنوات

## ✨ المميزات الجديدة:

### 🎯 إدارة القنوات المتقدمة:
- **تحديد قناة مخصصة** لكل سيرفر للرسائل التلقائية
- **حفظ الإعدادات** تلقائياً لكل سيرفر
- **إدارة متعددة السيرفرات** - كل سيرفر له إعداداته المستقلة

### 🆕 الكوماندات الجديدة:

#### `!تحديد_قناة` (للمشرفين)
```
!تحديد_قناة #اسم-القناة
!تحديد_قناة  # للقناة الحالية
```
- يحدد القناة التي سيرسل فيها البوت الرسائل التلقائية
- يتطلب صلاحية "إدارة القنوات"

#### `!إلغاء_قناة` (للمشرفين)
```
!إلغاء_قناة
```
- يلغي الرسائل التلقائية للسيرفر
- يتطلب صلاحية "إدارة القنوات"

#### `!معلومات_قناة` (للجميع)
```
!معلومات_قناة
```
- يعرض القناة المحددة حالياً
- يعرض أوقات الرسائل والإعدادات

## 🔧 التحسينات:

### 📊 قاعدة بيانات محلية:
- حفظ إعدادات كل سيرفر في `server_settings.json`
- استمرارية الإعدادات حتى بعد إعادة التشغيل
- نظام آمن وسريع للبيانات

### 🛡️ معالجة أخطاء محسنة:
- رسائل خطأ واضحة للصلاحيات
- التعامل مع القنوات المحذوفة
- حماية من الأخطاء في الإرسال

### 🌐 دعم متعدد السيرفرات:
- كل سيرفر له قناة مستقلة
- رسائل تلقائية لجميع السيرفرات المُعدة
- إدارة منفصلة لكل سيرفر

## 📋 قائمة الأوامر المحدثة:

### 🎯 أوامر الرسائل (للجميع):
- `!صباح` - رسالة صباحية فورية
- `!مساء` - رسالة مسائية فورية
- `!حب` - رسالة محبة
- `!تشجيع` - رسالة تشجيع
- `!مساعدة` - قائمة الأوامر الكاملة

### ⚙️ أوامر الإدارة (للمشرفين):
- `!تحديد_قناة` - تحديد قناة الرسائل التلقائية
- `!إلغاء_قناة` - إلغاء قناة الرسائل التلقائية
- `!احصائيات` - إحصائيات السيرفر

### 📋 أوامر المعلومات (للجميع):
- `!معلومات_قناة` - معلومات القناة الحالية

## 🚀 كيفية الاستخدام:

### للمشرفين:
1. **تحديد القناة:**
   ```
   !تحديد_قناة #عام
   ```

2. **التحقق من الإعدادات:**
   ```
   !معلومات_قناة
   ```

3. **الاستمتاع بالرسائل التلقائية!** 🎉

### للأعضاء:
- استخدم الأوامر العادية في أي وقت
- تحقق من معلومات القناة باستخدام `!معلومات_قناة`

## 🎯 الرسائل التلقائية:

### 🌅 رسائل الصباح (8:00 ص):
- رسالة ترحيب بالصباح
- نصيحة أو تشجيع يومي

### 🌙 رسائل المساء (8:00 م):
- رسالة مسائية دافئة
- تأمل أو تذكير إيجابي

### 💕 رسائل المحبة (كل ساعتين):
- رسائل إيجابية عشوائية
- نشر المحبة والإيجابية

### 🎉 ترحيب بالأعضاء الجدد:
- رسالة ترحيب تلقائية
- تتضمن صورة العضو

## 📁 الملفات الجديدة:

- `database.py` - نظام قاعدة البيانات
- `CHANNEL_COMMANDS.md` - دليل الكوماندات الجديدة
- `server_settings.json` - ملف حفظ الإعدادات (يُنشأ تلقائياً)
