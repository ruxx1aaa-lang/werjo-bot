# 📋 دليل الأوامر الإدارية - Werjo Bot

## 🎯 نظرة عامة

يحتوي Werjo Bot على مجموعة من الأوامر الإدارية التي تسمح للمشرفين بإدارة إعدادات البوت، خاصة تحديد القناة التي ستصل إليها الرسائل التلقائية والمقولات العشوائية.

## 🔐 متطلبات الصلاحيات

لاستخدام الأوامر الإدارية، يجب أن يكون لديك إحدى الصلاحيات التالية:
- **إدارة القنوات** (`manage_channels`) - للأوامر `!setchannel` و `!removechannel`
- **مشرف** (`administrator`) - لأمر `!debug`

## 📝 قائمة الأوامر الإدارية

### 1. `!setchannel` - تحديد قناة الرسائل

**الوصف:** يحدد القناة التي ستصل إليها المقولات العشوائية كل ساعة.

**الاستخدام:**
```
!setchannel                    # تحديد القناة الحالية
!setchannel #اسم-القناة       # تحديد قناة معينة
!setchannel @القناة           # تحديد قناة بالمنشن
```

**أمثلة:**
- `!setchannel` - يحدد القناة التي كتبت فيها الأمر
- `!setchannel #general` - يحدد قناة general
- `!setchannel #💬-دردشة` - يحدد قناة الدردشة

**الرد المتوقع:**
```
✅ تم تحديد القناة بنجاح!
📍 القناة: #general
🕐 ستصل المقولات العشوائية كل ساعة إلى هذه القناة
```

---

### 2. `!channelinfo` - معلومات القناة المحددة

**الوصف:** يعرض معلومات عن القناة المحددة حالياً للرسائل التلقائية.

**الاستخدام:**
```
!channelinfo
```

**الرد المتوقع (إذا كانت هناك قناة محددة):**
```
📋 معلومات القناة المحددة
📍 القناة: #general
🕐 المقولات العشوائية: كل ساعة
🆔 معرف القناة: 123456789012345678
```

**الرد المتوقع (إذا لم تكن هناك قناة محددة):**
```
❌ لا توجد قناة محددة!
استخدم !setchannel لتحديد قناة للرسائل التلقائية.
```

---

### 3. `!removechannel` - إلغاء قناة الرسائل

**الوصف:** يلغي القناة المحددة للرسائل التلقائية. بعد هذا الأمر، ستعود المقولات للإرسال في القنوات العامة.

**الاستخدام:**
```
!removechannel
```

**الرد المتوقع:**
```
✅ تم إلغاء قناة الرسائل التلقائية بنجاح!
لن تصل المقولات العشوائية بعد الآن.
```

---

### 4. `!debug` - معلومات التشخيص

**الوصف:** يعرض معلومات تفصيلية عن حالة البوت لمساعدة المشرفين في حل المشاكل.

**الاستخدام:**
```
!debug
```

**الرد المتوقع:**
```

🤖 معلومات البوت
الاسم: Werjo Bot
المعرف: 1234567890123456789
الخوادم: 1

🔐 صلاحيات البوت
إرسال الرسائل: ✅
إدارة الرسائل: ✅
الاتصال الصوتي: ✅
التحدث: ✅

📍 القناة المحددة
✅ #general (ID: 123456789012345678)

⏰ المهام التلقائية
المقولات العشوائية: 🟢 نشط
```

## 🚨 رسائل الخطأ الشائعة

### خطأ الصلاحيات
```
❌ خطأ في الصلاحيات
تحتاج إلى صلاحية إدارة القنوات لاستخدام هذا الأمر!
```
**الحل:** تأكد من أن لديك صلاحية "إدارة القنوات" في السيرفر.

### القناة غير موجودة
```
⚠️ القناة المحفوظة لم تعد موجودة!
استخدم !setchannel لتحديد قناة جديدة.
```
**الحل:** استخدم `!setchannel` لتحديد قناة جديدة.

### ملف الإعدادات تالف
```
❌ ملف الإعدادات تالف!
استخدم !setchannel لإعادة تحديد القناة.
```
**الحل:** استخدم `!setchannel` لإعادة إنشاء الإعدادات.

## 🔄 كيف يعمل النظام

1. **حفظ الإعدادات:** عند استخدام `!setchannel`، يتم حفظ معرف السيرفر ومعرف القناة في ملف `channel_settings.txt`
2. **المقولات التلقائية:** كل ساعة، يبحث البوت عن القناة المحددة ويرسل مقولة عشوائية
3. **القناة الاحتياطية:** إذا لم توجد قناة محددة أو فشل الإرسال، يبحث البوت عن قناة عامة تلقائياً

## 💡 نصائح للاستخدام

1. **تحديد قناة مخصصة:** استخدم `!setchannel` لتحديد قناة مخصصة للرسائل الإيجابية
2. **فحص الحالة:** استخدم `!channelinfo` للتأكد من أن القناة محددة بشكل صحيح
3. **حل المشاكل:** استخدم `!debug` إذا واجهت أي مشاكل مع البوت
4. **الصلاحيات:** تأكد من أن البوت لديه صلاحية الإرسال في القناة المحددة

## 🎯 الميزات التلقائية

- **المقولات العشوائية:** ترسل كل ساعة تلقائياً
- **الرسائل الصباحية والمسائية:** يمكن تفعيلها يدوياً بالأوامر
- **رسائل المحبة والتشجيع:** متاحة في أي وقت
