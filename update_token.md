# 🔄 تحديث التوكن

## خطوات الحصول على توكن جديد:

### 1. Discord Developer Portal:
1. اذهب إلى: https://discord.com/developers/applications
2. اختر تطبيقك أو أنشئ واحد جديد
3. اذهب إلى تبويب "Bot"
4. انقر "Reset Token"
5. انسخ التوكن الجديد

### 2. تحديث في Railway:
1. اذهب إلى مشروعك في Railway
2. تبويب "Variables"
3. حدث DISCORD_TOKEN بالتوكن الجديد

### 3. تحديث محلياً (للاختبار):
في ملف .env:
```
DISCORD_TOKEN=التوكن_الجديد_هنا
```

### 4. اختبار التوكن:
```bash
python test_token.py
```

## ملاحظات مهمة:
- التوكن حساس جداً - لا تشاركه مع أحد
- انسخ التوكن كاملاً بدون مسافات
- التوكن يبدأ عادة بـ MTU أو MTE أو NTU
- إذا أعدت إنشاء التوكن، التوكن القديم لن يعمل
